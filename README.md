# Superstore Profitability Insights Dashboard

## Project Overview
This project is developed as part of the ACC102 Mini Assignment. It is an interactive data product designed to assist Sales Managers in analyzing the profitability of the Superstore dataset. Instead of static reports, this tool allows users to filter by product categories and explore the relationship between sales volume and profit margins to identify underperforming areas.

## Features
- **Interactive Visualization**: Uses Plotly to provide dynamic charts (Bar charts and Scatter plots) that respond to user input.
- **Profitability Analysis**: Calculates custom 'Profit Margin' metrics to move beyond simple sales volume analysis.
- **User-Centric Design**: Built with Streamlit, focusing on a clean, decision-oriented interface.

## Prerequisites
To run this project, you need to have Python installed. You will also need the following libraries:
- pandas
- streamlit
- plotly
